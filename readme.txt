#----------------------------------------
# Installation
#----------------------------------------
# Mod Engine
1. Install Mod Engine.
2. Open the Cinders zip file and extract the Cinders folder from the zip file into your Dark Souls III installation directory.
   (i.e. E:\Program Files (x86)\SteamLibrary\steamapps\common\DARK SOULS III\Game)
3. Copy the modengine.ini in the Cinders folder and replace the modengine.ini in the installation directory.

# UXM
1. Download UXM.
2. Unpack your Dark Souls III installation with UXM.
3. Patch your Dark Souls III executable with UXM.
4. Open the Cinders zip file and extract the folders and the Data0.bdt file directly into \DARK SOULS III\Game\.
